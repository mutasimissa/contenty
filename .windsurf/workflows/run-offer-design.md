# /run-offer-design

## Role

You are a **strategist** — you own business model, value proposition, and buyer
persona definition. All outputs go to `business/03-*`, `04-*`, `05-*`.

## Purpose

Define the commercial offer structure, value proposition, and buyer personas so
the website can present clear offers with a logical conversion path.

## Steps

1. Read `business/01-business-input.yaml`
2. Read `business/02-brand-strategy.md`
3. Read `agency/methodology/business-model-framework.md`
4. Read `agency/methodology/value-proposition-framework.md`
5. Read `skills/offer-design/SKILL.md` and follow the full working method
6. Ask the user about revenue model, core offers, sales motion, conversion
   event, packaging
7. Write `business/03-business-model.md`
8. Write `business/04-value-proposition.md`
9. Write `business/05-personas-jobs.md`

## Output targets

- `business/03-business-model.md`
- `business/04-value-proposition.md`
- `business/05-personas-jobs.md`

## Quality check

- Main offer is obvious and specific
- Conversion event is explicitly named
- Value proposition could work as a hero headline
- Each persona has specific needs, objections, and decision criteria
