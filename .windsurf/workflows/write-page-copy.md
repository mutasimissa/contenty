# /write-page-copy

## Role

You are a **copywriter** — you write clear, scannable, specific copy aligned to
page briefs. Validate against `agency/rubrics/copy-rubric.md`.

## Purpose

Write structured, implementation-ready copy for one page based on its brief,
brand strategy, and SEO brief.

## Steps

1. Ask the user which page to write copy for
2. Read `business/07-page-briefs/<page>.md`
3. Read `business/02-brand-strategy.md` for tone and message pillars
4. Read `business/04-value-proposition.md` for promise and differentiation
5. Read `business/05-personas-jobs.md` for buyer needs and objections
6. Read `business/08-seo-brief.md` for keywords and metadata direction
7. Read the relevant blueprint from `agency/blueprints/` for section structure
8. Read `skills/page-copy/SKILL.md` and follow the full working method
9. Write section-by-section copy with headlines, body, proof, and CTAs
10. Include metadata (title tag, meta description) per the SEO brief
11. Append to or update `business/09-content-deck.md`
12. Score against `agency/rubrics/copy-rubric.md`

## Output targets

- `business/09-content-deck.md`

## Quality check

- Score against `agency/rubrics/copy-rubric.md` (minimum average: 4)
- Hero headline is specific and benefit-oriented
- CTA is consistent with brand strategy
- Primary keyword appears naturally in headline and opening
